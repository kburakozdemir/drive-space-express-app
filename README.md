# Notes

Node.js, Express hosted on free tier Azure web app and fetching data from Hasura.

Visit [this url](https://nodetestexpress.azurewebsites.net/) to see the app.

```bash
# Node v18.20.3
npm install
```

```bash
npm run start # http://localhost:4000
```
